## Gramatyka

### Zaimki osobowe
| Zaimek | Znaczenie |
|--------|-----------|
| ich | ja |
| du | ty (nieformalne) |
| er / sie / es | on / ona / ono |
| wir | my |
| ihr | wy |
| Sie | Pan/Pani (formalne) |
| sie | oni/one |

### Koniugacja "sein" (być) — czas teraźniejszy
| Osoba | Forma | Znaczenie |
|-------|-------|-----------|
| ich | **bin** | jestem |
| du | **bist** | jesteś |
| er/sie/es | **ist** | jest |
| wir | **sind** | jesteśmy |
| ihr | **seid** | jesteście |
| sie/Sie | **sind** | są / jest (Pan/i) |

### Pytania o imię
- **Formalne:** Wie heißen Sie? → Ich heiße...
- **Nieformalne:** Wie heißt du? → Ich heiße...

## Przykładowy dialog
> **A:** Guten Tag! Wie heißen Sie?
> **B:** Guten Tag! Ich heiße Andrzej. Und Sie?
> **A:** Ich heiße Maria. Woher kommen Sie?
> **B:** Ich komme aus Polen. Freut mich!
> **A:** Freut mich auch!
